# Top, ps, jobs, jill
리눅스에서 프로세스를 확인하고 제어할 수 있는 주요 명령어를 정리

## Top 명령어
실시간으로 시스템 프로세스와 자원(CPU, 메모리) 사용량을 모니터링

## 주요 옵션
| 옵션 | 설명 |
|------|------|
| q | 반응 속도 빠르게 |
| c | 실행 경로/명령 전체보기 |
| k | 프로세스 종료 (PID 입력) |
| h | 도움말 표시 |

## 사용 예시
top

### ps 명령어
현재 실행 중인 프로세스를 확인
자주 실행하는 옵션: 'ps -e', 'ps -ef', 'ps aux'
